Learning Single and Cross Network Node Representation via Radial Random Walk

Abstract

Network embedding for vector-like node representation in networks has recently attracted intensive attention. Most existing methods are focused on single-net embedding, and the current single-network embedding methods can hardly have the two desirable properties at the same time: 1) scalability for large-scale networks; 2) ability to incorporate both multi-order proximity and BFS/DFS structure information for more effective representation learning. On the other hand, for the emerging and practically important problem of cross-network embedding and its application like network alignment, most existing works train each graph's embedding separately and learn an alignment function afterwards, which is not end-to-end hence can be non-optimal for overall optimization.
In this paper, we propose Rep-Align, a framework for both single-net and cross-net embedding. For single-net, we first devise an unsupervised shallow embedding model that jointly models the 1st-order proximity and 2nd-order proximity via Laplacian Eigenmaps and probabilistic model, respectively. Also the higher-order (3rd or higher) proximity is explicitly modeled via an objective based on weighted multi-hop adjacency matrix. Moreover, structure preservation that accounts for both local and global structure is also considered in the paper. In particular, the higher-order proximity as well as the structure model can be solved by our proposed Radial Random Walk strategy, which is fast and theoretically sound based on our analysis.
For the case of cross-net embedding, we propose a weakly-supervised method which performs joint node embedding and network alignment based on the given seed matchings. The cross-net embedding model adopts Gaussian Mixture Model (GMM) for constraints between seed matching nodes or estimated matching nodes. The Expectation-Maximization (EM) framework is adopted to effectively boost the seed matching correspondence between two networks. Experiments are performed on real-world datasets on tasks including node multi-label classification, network reconstruction, and network alignment. The results show the state of the art performance of the proposed methods compared with the peer methods for both single-net embedding and network alignment.
